Description of Dataset

Landsat-8 datasets

These are five-class datasets containing labelled patterns collected from Landsat-8 satellite images captured over the Indian provinces of West Bengal (WB) and Gujarat (GJ). The datasets consist of 11 band values in Columns 3-13 as features (the first two columns are the pixel coordinates) for 1858 and 1578 pixels labelled as one of the following classes for WB and GJ datasets, respectively.

  • Class-1: Crop
  • Class-2: Forest
  • Class-3: Soil
  • Class-4: Urban
  • Class-5: Water

    • Column 14 contains the class label

Please cite the following paper when publishing results that use this dataset:
S. Chakraborty, M. Roy, and F. Melgani, "Semi-supervised Two-level Fusion based Auto-encoded Approach for Low-cost Domain Adaptation of Remotely Sensed Images" IEEE Geoscience and Remote Sensing letters, 16: 1041-1045, 2019

Ikonos-2 datasets

These are seven-class datasets containing labelled patterns collected from Ikonos-2 satellite images captured over the Indian cities of Pondicherry (PO) and Chennai (CH). The datasets consist of RGB values in Columns 3-5 as features (the first two columns are the pixel coordinates) for 879 and 1229 pixels as one of the following classes for PO and CH datasets, respectively.

  • Class-1: Building
  • Class-2: Field
  • Class-3: Freshwater
  • Class-4: Road
  • Class-5: Sea
  • Class-6: Soil
  • Class-6: Tree

    • Column 5 contains the class label

Please cite the following paper when publishing results that use this dataset:
S. Chakraborty, M. Roy, and F. Melgani, "Semi-supervised Two-level Fusion based Auto-encoded Approach for Low-cost Domain Adaptation of Remotely Sensed Images" IEEE Geoscience and Remote Sensing letters, 16: 1041-1045, 2019
